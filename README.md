# Tumblr Growth Intelligence Platform

AI-powered analytics platform designed to help Tumblr indie music creators understand audience engagement, optimize captions, improve posting timing, and grow faster using data.

## Project Overview

This project transforms Tumblr post data into actionable creator intelligence through:

- Predictive engagement scoring
- Caption performance analysis
- Best posting hour insights
- Tag strategy optimization
- Emotional tone analysis
- Growth recommendations dashboard

## Tech Stack

- Python
- Pandas
- Scikit-learn
- Streamlit
- Plotly
- Joblib

## Features

### Executive Overview
Business-ready summary with KPIs and creator recommendations.

### Analytics Dashboard
Interactive visual dashboards showing:

- Best time to post
- Caption length impact
- Tag usage vs engagement
- Emotional tone vs success
- Community tag trends

### Strategic Post Simulator
Enter a caption, tags, and posting time to predict engagement probability before publishing.

### Growth Playbook
Practical recommendations creators can apply immediately.

### Consulting Summary
Business-focused insights and final recommendations.

## Files Included

- app.py -> Main Streamlit application
- tumblr_features.csv -> Processed analytics dataset
- tumblr_model.pkl -> Trained machine learning model
- requirements.txt -> Required packages

## Business Impact

Helps creators move from guessing to data-driven publishing decisions.
